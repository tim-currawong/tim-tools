# Tim-Tools

Tim's Personal Toolkit - A collection of useful calculators and converters.

## Features

- **Battery Configuration Calculator** - Configure series/parallel battery packs and calculate total capacity, voltage, and discharge limits
- **Wire Gauge Calculator** - Select the right wire gauge based on current draw and cable length to minimize voltage drop
- **Unit Converter** - Fast bidirectional converter for length, mass, force, torque, pressure, temperature, area, volume, energy, and power

## Technology

- Pure vanilla JavaScript (no frameworks)
- No build process - runs directly in browsers
- Client-side only - no backend required
- Dark/light theme support
- Mobile responsive

## Getting Started

Simply open `index.html` in your browser. No installation or setup required.
